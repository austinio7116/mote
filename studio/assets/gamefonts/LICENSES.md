# Bundled starter fonts

These TrueType fonts ship with Mote Studio as one-click starting points in the Font
tab. Each has been subset to printable ASCII (U+0020–U+007E). All are freely
redistributable under the licenses below; their copyright and license notices are
retained here.

| `Mono.ttf`     | DejaVu Sans Mono       | Bitstream Vera License + public-domain additions (permissive) |
